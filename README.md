# Diabetes-Analysis-and-Prediction

**Overview:**

This project focuses on analyzing diabetes-related health data and building a machine learning model to predict whether a person is diabetic based on medical attributes.

**Dataset:**

Diabetes dataset (medical measurements)
Features include:
Glucose level
Blood pressure
BMI
Age
Insulin
Pregnancies

**Tools:**

Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
Jupyter Notebook

**Analysis Performed:**

Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Feature selection
Model training and evaluation
Diabetes prediction using ML algorithm

**Output:**

Visual insights from data analysis
,Prediction of diabetic vs non-diabetic patients.
